# Sprint Board Kurulumu ve Kartlar

GitHub Projects ile 15 dakikada kurulum (repoya sekme olarak yapışır, ayrı araç gerekmez):

1. Repo sayfası → **Projects** sekmesi → **New project** → "Board" şablonu → ad: `Çarpan Sprint Board`
2. Kolonları düzenle: `Backlog · To Do · In Progress · Review · Done`
3. Aşağıdaki kartları ekle (**+ Add item** → başlığı yapıştır) ve kolonuna sürükle
4. Sağ üstten **⋯ → Settings → Visibility: Public** yap (jüri görebilsin)
5. Board linkini ana `README.md`'deki iki placeholder'a yaz
6. Board'un ekran görüntüsünü al → `ProjectManagement/Sprint2Documents/screenshots/board-sprint2.png`
