# Daily Scrum Notları — Sprint 3'te geçerli format

Her gün için `2026-07-XX.md` dosyası (ya da yazışma ekran görüntüsü). Format:

```markdown
# Daily — 21 Temmuz 2026

| Üye | Dün | Bugün | Engel |
|---|---|---|---|
| Görkem | ... | ... | - |
| Bahar | ... | ... | - |
| Doğa | ... | ... | - |
| Emir | ... | ... | - |
```

Sprint 2 notu: bu sprintte daily ritmi düzenli işlemedi (bkz. sprint raporu, retrospektif) —
koordinasyon PR açıklamaları ve birebir mesajlaşmayla yürüdü. Sprint 3'te her akşam kısa
yazılı daily bu klasöre işlenecek.
